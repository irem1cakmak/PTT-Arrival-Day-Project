
# PTT Kargo Varış Günü Projesi / PTT Arrival Day Project

## Proje Açıklaması / Project Description

PTT Kargo Varış Günü projesi, staj kapsamında geliştirilmiş bir Java tabanlı web uygulamasıdır.  
Projede PostgreSQL veritabanı ve Spring MVC framework’ü kullanılmıştır.  
Katmanlı mimari (Controller, Service, Repository) ile tasarlanmış olup veri akışı DTO kullanılarak optimize edilmiştir.

The PTT Arrival Day Project was developed during my internship as a Java-based web application.  
The project uses PostgreSQL as the database and Spring MVC as the framework.  
The architecture follows a layered design (Controller, Service, Repository), and data flow is optimized using DTOs.

---

## Kullanılan Teknolojiler / Technologies Used

- Programlama Dili / Programming Language: Java  
- Framework: Spring MVC  
- Veritabanı / Database: PostgreSQL  
- Proje Yönetimi / Project Management: Maven  
- Araçlar / Tools: IntelliJ IDEA, Postman  

---

## API Entegrasyonu / API Integration

Projede hem REST hem de SOAP API entegrasyonu gerçekleştirilmiştir.  
Web servisleri aracılığıyla veri alışverişi sağlanmış ve tüm süreçler raporlanmıştır.

The project implements both REST and SOAP APIs for communication between services.  
Data exchange is handled via web services, and all processes are documented.
