PTT Arrival Day Project
Project Description

The PTT Arrival Day Project was developed during my internship as a Java-based web application. The project uses PostgreSQL as the database and Spring MVC as the framework. The architecture follows a layered design (Controller, Service, Repository), and data flow is optimized using DTOs.

Technologies Used
Programming Language: Java
Framework: Spring MVC
Database: PostgreSQL
Project Management: Maven
Tools: IntelliJ IDEA, Postman
API Integration

The project implements both REST and SOAP APIs for communication between services. Data exchange is handled via web services, and all processes are documented.

Project Process
Designed using Spring MVC architecture and layered structure
Managed and integrated data using PostgreSQL
Implemented REST and SOAP API integrations
Tested API endpoints using Postman
Prepared a detailed PDF documentation explaining the step-by-step project workflow
